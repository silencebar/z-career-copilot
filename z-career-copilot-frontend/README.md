# Career Copilot Frontend

Career Copilot 前端基于 Vue 3 + Vite 构建，提供 AI 职场沟通与面试辅导助手和超级智能体两个页面。

## 功能

- 职场沟通与面试辅导：支持面试准备、简历表达、汇报话术、同事协作和领导沟通建议。
- 超级智能体：支持学习路线规划、任务拆解、行动清单和工具增强型问答。

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 后端接口

- `/api/ai/career/chat/sse`：职场沟通与面试辅导助手聊天接口
- `/api/ai/agent/chat`：超级智能体聊天接口

接口路径暂时保留，以兼容后端现有路由。
